VI:visual interface

在终端使用vimtutor可以查看vim的所有命令

VIM:
	-支持多级撤销（通过按字母U来撤销上一步的操作）
	-可以跨平台运行（windows linux都可以使用）
	-语法高亮
	-图形界面

vim的操作模式：
	命令模式 command mode
	插入模式 insert mode
	尾行模式 last line mode
	
	vim abc
	（如果文件不存在则自动创建一个文件，存在则打开）
	dd 	删除改行
	i  	切换插入模式
	Esc 切换到尾行模式
	
vim 编辑器的命令格式
	vim abc			打开文件abc
	vim + abc		打开文件之后把光标定位到文件abc的最后一行
	vim +3 abc		打开文件之后把光标定位到文件abc的第三行
					（当数字大于最大行数时，光标将被定位到最后一行）
	vim +/xxx abc 	打开文件后将光标定位到xxx第一次出现的那一行
					（如果xxx出现多次，可以按n来切换）
	vim aa bb cc	一次性创建或打开多个文件，如果文件不存在则创建，存在则打开
					在尾行模式下输入‘：n’可以向后切换文件
					在尾行模式下输入‘：N’或'：prev'可以向前切换文件

常用指令
	尾行模式常用指令：
		：w 	把文件内容写入到硬盘中
		：q 	退出vim编辑器
		:!		强制执行
		：ls	列出当前编辑器打开的所有文件
		：n		向前切换文件
		:N		向后切换文件
		:xx		xx是一个数字，可以把光标定位到第xx行
		/xxx	从光标开始向后搜索xxx，并且把光标定位到xxx第一次出现的位置
		？xxx	从光标开始向前搜索xxx，并且把光标定位到xxx第一次出现的位置
	命令模式常用指令:
		h			光标左移
		j			光标下移
		k			光标上移
		l			光标右移
		ctrl+f		向下翻页（front）
		ctrl+b		向上翻页（back）
		ctrl+d		向下翻半页（down）
		ctrl+u		向上翻半页（up）
		dd			删除光标所在行
		o			在光标所在行的下方插入一行并切换到输入模式
		yy			复制光标所在行
		p（小写）	在光标所在行的下方粘贴
		P（大写）	在光标所在行的上方粘贴
